#LCN v.b51 SETUP values

LEGEND

  • [default value, if not specified]
  • [a valid value]

SETUP


autoplay

Toggles between manual mode (PgUp) and automatical map rotation
  • false - manual rotation
  • true - automatic rotation, maps cycle by themselves

autorespawn

Toggles between automatic respawn and the default death which needs to be respawned with !r
  • false - if mice die, they need to be revived with !r
  • true - impossible to die, respaws automatically

challengemode

Switches the mode of the module
  • build - builders' battle/The Challenge, free mode
  • pkg - package rotation which is specified by the second argument of !c. To see the list of valid map rotations, click here.
  • survival - survival challenge
  • thirty - 30 second challenge rotation with countdown
  • vanilla - shaman-related contest with vanilla rotation

debug

Toggles permission to use !debug
  • false - module works as usual
  • true - !debug command is available

lang

Changes language
  • lv - Latvian
  • en - English
  • de - German

logcallbacks

Toggles the visibility of actions in #lua
  • false - actions are shown only in HostCore
  • true - actions are shown in #lua like commands

lsimage

Changes the loading screen's left side's image
  • 16f09800dd9.png - La Cros Nostra logo

nocontrast

Toggles between visibility of contrast in UtilCore
  • false - UtilCore uses contrast
  • true - UtilCore does not use contrast, mandatory for builders' battle/The Challenge

panelpos

Changes the position of the control panel
  • 1 - left side, offscreen
  • 2 - left side, onscreen
  • 3 - right side, offscreen
  • 4 - right side, onscreen

title

Changes the title of the module
  • lacrosnostra - tribe name "La Cros Nostra"

theme

Changes the theme of the module
  • modern - default blue theme
  • classic - theme reminiscent of the one used from v.b26 to v.b40 (not finished yet)
  • oldclassic - theme reminiscent of the one used from v.b1 to v.b25 (not finished yet)
