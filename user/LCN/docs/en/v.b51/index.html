#LCN v.b51 documentation

LEGEND

  • [mandatory argument]
  • [optional argument]

CONTESTANT COMMANDS

  • !i - opens settings
  • !pr [name] - opens player's technical profile
  • !spawn [id][x][y][angle][x acceleration][y acceleration][is transparent?] - summons an object

HELPER COMMANDS

  • !bc [text] - shows yellow text in the upper interface for every player (without HTML)
  • !ce [name|all] - gives cheese to the player
  • !ce* [name|all] - takes the cheese from the player
  • !clear - clears all objects from the map
  • !f [name|all] - freezes the player
  • !f* [name|all] - unfreezes the player
  • !hc - opens HostCore
  • !meep [name|all] - grants meep ability to the player
  • !r [name|all] - respawns the player
  • !s [name|all] - makes the player a shaman
  • !s* [name|all] - takes away the shaman powers from the player
  • !size [name|all][size] - changes the player's size
  • !smode [name|all][mode] - makes the player a shaman with skills. If the second argument is specified, changes the shaman mode of the player.

HOST COMMANDS

  • !add [name|all][value] - increments/decrements the player's score by the specified value in the scoreboard
  • !barrier [parts][map's length] - splits the map in parts using barriers
  • !bc* [text] - shows green text in the upper interface for every player (with HTML)
  • !c [mode][pkg map rotation] - begins a contest
  • !cd [name][background transparency][background color][message x][message y][reason] - disqualifies the player with a custom disqualification interface
  • !col [name|all] - colors the player's name
  • !d [name][reason] - disqualifies the player with ability to spectate
  • !d* [name][reason] - disqualifies the player without the ability to spectate
  • !debug [function][arg1][arg2][arg3][arg4][arg5][arg6][arg7][arg8][arg9][arg10][arg11][arg12][arg13][arg14] - runs a function from the code, only works if SETUP.debug is true.
  • !defcol [tags] - definē jaunu HTML krāsu tagu
  • !end - shows end map, ends contest
  • !end* - shows end map and shows a feedback window for everyone
  • !g [id][type][x][y][width][height][restitution][friction][rotation][is dynamic?][mass] - creates a ground with collision
  • !g* [id][type][x][y][width][height][restitution][friction][rotation][is dynamic?][mass] - creates a ground without collision
  • !hc [flags...] - opens HostCore
  • !hc* - opens HostCore in tabbed view
  • !kill [name|all] - kills the player
  • !link [name][name2|all] - links players as soulmates
  • !link* [name][name2|all] - unlinks players as soulmates
  • !map [mgoc][width][height][background][ground type][ground color] - loads a custom map
  • !md [id][x1][y1][x2][y2][width] - draws a line
  • !md* [id][x1][y1][x2][y2][width][color][transparency] - draws a line
  • !msg [name][message] - sends the player a private message in the upper interface
  • !np [code] - plays the specified map or a random map from the specified category
  • !np* [code] - plays the specified maps mirrored version or a random mirrored map from the specified category
  • !p [id][type][text] - shows a popup
  • !play [name|all][id][extra data] - makes the player do an emote
  • !r* [name|all][x][y] - respawns the player in the specified coordinates
  • !rmtxt [id][flags...] - removes a text area
  • !role [action][role][name] - grants/strips the player of a role
  • !score [name|all][value] - sets the player's score to the specified value in the scoreboard
  • !settp [id][x][y] - sets a teleportation destination
  • !sname [text] - changes shaman's name in the information bar
  • !sp [name] - makes the player a spectator
  • !spawn* [count][id][x][y][angle][x acceleration][y acceleration][is transparent?] - summons multiple objects
  • !startfrom [round] - begins the contest from the specified round
  • !test - plays a survival test map
  • !tfm [name|all] - allows the player to transform
  • !tfm* [name|all] - disallows the player to transform
  • !tr [id] - shows a translation in the HostCore interface
  • !txt [id][x][y][color][text][flags...] - creates a text area
  • !uc [text] - shows text in the lower interface for every player (with HTML)
  • !uc* [text] - shows highlighted text in the lower interface for every player (with HTML)
