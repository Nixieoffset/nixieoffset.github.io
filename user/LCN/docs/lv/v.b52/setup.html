SETUP


allowemptyfeedback

Deaktivizē tukšu atsauksmju bloķēšanu
  • false - tukšas atsauksmes tiek bloķētas
  • true - tukšas atsauksmes netiek bloķētas

autoplay

Pārslēdzas starp manuālo režīmu (PgUp) un automātisku mapju pārslēgšanos
  • false - manuāla rotācija
  • true - automātiska rotācija, mapes pārslēdzas pašas

autorespawn

Pārslēdzas starp automātisku atdzīvošanos nāves gadījumā vai standarta nāvi, kura jāatdzīvina ar !r
  • false - ja peles nomirst, tās jāatdzīvina ar !r
  • true - nav iespējams nomirt, automātiski atdzīvojas

challengemode

Pārslēdz moduļa režīmu
  • build - būvdarbu cīņas/izaicinājumu izaicinājums, tukšgaita
  • pkg - pakotnes rotācija, to nosaka !c otrais parametrs. Mapju rotācijas skatīt šeit.
  • survival - izdzīvošanas challenge
  • thirty - 30 sekunžu challenge mapes ar countdown
  • vanilla - šamaņa konkursi ar vanilla rotāciju
  • mono - vienzemju challenge mapes ar countdown

debug

Aktivizē debug režīmu
  • false - modulis darbojas normāli
  • true - kļūst pieejama !debug komanda un parādās debug logs

disablesuperpowers

Deaktivizē superspējas
  • false - superspējas darbojas normāli
  • true - superspējas pieejamas tikai, ja tās manuāli iedotas ar !action

elevatedmode

Aktivizē privileģēto (tikai FunCorp un moduļu komanda) režīmu
  • false - modulis darbojas normāli
  • true - !uc un !uc* rādās čatā, nevis UtilCore interfeisā

lang

Maina valodu
  • lv - latviešu
  • en - angļu
  • de - vācu
  • debug - debug

logcallbacks

Pārslēdz darbību rādīšanu #lua
  • false - darbības tiek rādītas tikai izmantojot HostCore
  • true - darbības tiek rādītas #lua tāpat kā komandas

lsimage

Nomaina sākuma ekrāna kreisā sāna logo
  • 16f09800dd9.png - La Cros Nostra logo

nocontrast

Pārslēdzas starp kontrasta režīmiem
  • false - utilcore lieto kontrastu
  • true - utilcore nelieto kontrastu, obligāti būvdarbu cīņām/izaicinājumu izaicinājumam

panelpos

Maina paneļa atrašanās vietu
  • 1 - kreisā puse, ārpus ekrāna
  • 2 - kreisā puse, iekš ekrāna
  • 3 - labā puse, ārpus ekrāna
  • 4 - labā puse, iekš ekrāna

title

Maina moduļa vārdu
  • lacrosnostra - "La Cros Nostra" cilts nosaukums

theme

Maina moduļa dizainu
  • modern - standarta zilais dizains
  • classic - dizains, kas līdzīgs v.b26-v.b40 dizainam (nav pabeigts)
  • oldclassic - dizains, kas līdzīgs v.b1-v.b25 dizainam (nav pabeigts)
